Combinacions de selectors

Què són els combinadors?

Són la combinació de selectors CSS senzills. Què vol dir això? Significa que el selector de classes, el selector d'identificació o fins i tot els selectors d'etiquetes, tot en CSS són selectors simples i quan fem servir més d'1 selector es fa un selector combinador.

Per exemple:
div i p són selectors simples, però junts div p són ara selectors combinadors.
Bàsicament, el selector combinador defineix una relació entre els selectors.

  1. Selector descendent

    Bàsicament selecciona tots els elements dins o descendents de l'element especificat. Per exemple

            div p {
          color : red;
       }

    Colorejarà/estilitzarà tots els elements <p> dins dels elements <div> inclosos tots els elements <p> imbricats. Els elements imbricats són aquells que estan imbricats en un contenidor com... 

            <div>
          <p>Paràgraf</p>
          <h1><p>Paràgraf inbricat</p><h1>
        </div>

  2. Selector de fills ( > )

    Selecciona tots els elements dins de l'element especificat, excepte els elements imbricats. Per exemple, 
            div > p {
          color: red;
        }

    Per tant, l'exemple anterior seleccionarà tots els elements <p> de l'element <div> però no afectarà els elements <p> imbricats.

  3. Selector de germans adjacents ( + )

    El selector de germans adjacents selecciona els elements just després de l'element especificat. Per exemple,

            div + p {
          color: red;
        }

    Per tant, el codi anterior només seleccionarà l'element <p> que està just després de l'element <div>.

  4. Selector general de germans ( ~ )

    Aquest és l'últim i selecciona tots els elements després de l'element especificat.
    Per exemple,

          div ~ p {
        color: red;
      }

    L'exemple anterior seleccionarà tots els elements <p> després de l'element <div>.
